# Contributing to Pitwall

This repository is under the GNU General Public License v3.0 for the purpose of keeping as close of a community/collaborative environment around the original software as possible. 

All contributions to the project are welcome, including major refactoring, logical and performance improvements.

No feature is too large or too small to be added. If it's quality and a good improvement, create a PR!

If you want to contribute, but don't know where to start, check out (todo.md)[https://github.com/Ethan-Ka/Pitwall/blob/main/todo.md]

This project is meant to be accessible to those who want to use it, so electron has been elected for its cross platform support.

# Running the project

`start.bat` will include necessary commands to run the project. You can run them by executing the batch file, or running the commands if you're using MacOS or Linux

# Source Code

The project is structured as such
```
/pitwall/src
    /assets - app icons etc
    /components - reused and larger components of the app
    /hooks - self explanitory
    /lib - library of colors and constants
    /screens - different screens(just onboarding for now)
    /store - storage related code
    /types - default electron folder 
    /widgets - built in widgets widgets
```
